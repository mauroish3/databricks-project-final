# Trabajo Final – Ingeniería de Datos con Databricks

## 1) Descripción
Este repositorio contiene el desarrollo del trabajo final del curso de Ingeniería de Datos con Databricks. El enfoque del proyecto es levantar un flujo completo de procesamiento con Spark: configuración, ingesta, validaciones, transformaciones y consultas de análisis, dejando la ejecución separada por notebooks para facilitar su revisión.

Se trabajó con dos datasets: **ecommerce** y **automobile**.

---

## 2) Tecnologías utilizadas
- Databricks (Apache Spark)
- PySpark / Spark SQL
- GitHub (versionamiento del entregable)
